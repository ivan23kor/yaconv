# Installation

1. Install [blis](git@github.com:flame/blis.git)
1. Create symlinks to BLIS config files:
```bash
ln -s path/to/BLIS/config.mk config.mk
ln -s path/to/BLIS/config/ARCH/make_defs.mk make_defs.mk
```
